Инструкция для сборки и запуска программы:
1. Скачать проект и открыть его в Visual Studio.
2. Собрать проект в Visual Studio.
3. Перейти по пути: FolderGuide\FolderGuide\bin\Debug\ и поместить туда файл FolderDB.accdb который лежит в корневой папке репозитория.

 Инструкция по использованию:
1. На файлы и папки можно нажать ПКМ для вызова контекстного меню.
2. Для прерывания загрузки выбранной папки нужно нажать на кнопку "Открыть папку".
3. Для выбора нескольких файлов можно удерживать клавишу "Shift" или зажать клавишу "Ctrl".
